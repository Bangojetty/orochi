# Orochi

A Swiss-army knife of small tools for my work workflows. Each tool is meant to be
dead simple and dependency-free — open an HTML file and go.

## Tools

### Tracker (`index.html`)

A counter with a settable hotkey.

- **Count up** — press the hotkey (default `Space`) or click **Count +1**.
- **Rebind** — click **Rebind**, then press any key to set it as the hotkey (`Esc` cancels).
- **Reset** — set the counter back to `0`.

The count and your chosen hotkey persist in the browser via `localStorage`.

## Usage

Just open `index.html` in any browser. No build step, no dependencies.
